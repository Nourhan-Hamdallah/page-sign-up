🔑 Elegant Login & Registration Page
A clean, minimalist, and responsive Login and Account Creation interface. This project focuses on semantic HTML5 form structuring and custom CSS3 styling to deliver a smooth user experience for authentication flows.

🚀 Features
User Authentication UI: Clean input fields dedicated to gathering user credentials (Username & Email).
Form Validation: Uses native HTML5 required attributes to ensure users don't submit empty fields.
Navigation Flow: Includes an anchor link (create new acc) redirecting users to a secondary onboarding registration view (nextadge.html).
Modern Form Layout: Wrapped fields in structured container blocks (.one) allowing for isolated component styling, focus effects, and spacing control.
🛠️ Tech Stack
HTML5: Semantic architecture for user data collection form inputs, layout wrappers, and hyperlinks.
CSS3: Custom styles handling:
Centered layout card designs (Flexbox/Grid alignments).
Input field typography, borders, active focus triggers, and custom placeholder adjustments.
Interactive states for actions like button active clicks or links hover indicators.
📁 File Structure
├── index.html       # The main Login layout page
├── style.css        # The custom CSS styling rulesheet
└── nextadge.html    # The secondary registration target flow page
